# 極大可能社会選択ルールにおける障害と変形について：計算実験

本リポジトリは、日本オペレーションズ・リサーチ学会 2026年春季研究発表会にて報告された論文「極大可能社会選択ルールにおける障害と変形について：計算実験」の補遺資料および計算プログラムを公開するものです 。

## 1. 概要

アローの選好集計（ただし非制限性公理のみ緩和）を、二項的な選好集計ルール（IIA条件）の局所的な判断（パレート条件）から大域的に整合させる試みと解釈します。
また非推移性をその際の位相幾何学的障害として捉え直します。
すなわち層理論（Čech 1-コホモロジー）へ翻訳し、非推移性の原因となるプロフィールたちを「障害集合」として特定します。
この障害を除去したドメイン上では、実際に独裁ではない一貫した社会選好（極大可能ルール）が構成可能であることを計算実験により示します 。

## 2. 理論的背景

**コホモロジー障害**: 社会選好の非推移性をコボウンダリ  （ホロノミー）として同定します 。
**障害の特定**: 非推移性を生む三項サブプロフィールをPrologを使って特定し、いわば決定性含意のフローにおける極小カットセットとして除去します。
**可能性定理**: 障害集合(非推移性を生じるプロフィール)を除去したのちは 局所的な1-コチェインは大域的な1-コサイクルへと延長可能です（Alexander双対性のとして解釈）。
**不可能性定理**: 障害集合が空集合であることを要求すると大域的な1-コサイクルへと延長可能なのは独裁だけです（Arrowの定理の解釈）。

## 4. プログラムの構成

本リポジトリに含まれるコードの詳細は以下の通りです。

* `sct2025y2.pl`: 2項集計マップを生成してその障害を特定するための Prolog 実装コード （旧バージョンや2026は参考まで）。
* `lib2025.pl`: ヘルパープログラム集。sct2025y2と合わせて図1(a)-(d) に示された元データを生成できます。
* `njmk_max_poss...`: 極大ルール。
* `njmk_min_cut...`: 極小障害集合。

以下は直接アブストラクト内では参照していませんが，関連する結果を出力するものです．
* `arrow2024x-29Nov.pl`:プロフィールベースの選好ルール生成。集計ルールの極大性を検証するために使います．
* `alt5swf.pl`:5代替案で生成された集計ルールです．上記のコードを用いた，まだプロフィールベースでの生成です．
* `alt5swf_maxposs_verify.png`:上記の検証を行ったときのスクショです．
* `logical_scf_Nov_2025.pl`:極大耐戦略ルールの変形を検証するために使います．kindo2018/prologのコードと同等です．
* `network_draw.html`:決定性含意の視覚化に使います．csvファイルでネットワークデータをブラウザ内に読み込みます．
* `deform.pl, detect_decisive_channel.pl`:上記のルール変形や決定性含意の分析に使う述語です．
